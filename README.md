# MoviesApp
## Feature - 
1. Displays a List of Movies fetched from the api <a href="https://www.themoviedb.org/">themoviedb</a>
2. Offline caching by downloading images to File Manager and Storing Data to Core Data.

## Technologies -
1. Xcode
2. Swift
3. SwiftUI
4. Core Data
